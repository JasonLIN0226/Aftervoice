# After Distortion

Single-page Next.js artwork exploring what remains of a sentence after it enters a medium.

## Local run

1. Install dependencies with `npm install`
2. Copy `.env.example` to `.env.local`
3. Add `DEEPSEEK_API_KEY` if you want live DeepSeek transformations
4. Run `npm run dev`

If no API key is present, the app uses a deterministic local fallback transformation so it still works offline.
